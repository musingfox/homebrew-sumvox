# Homebrew Tap for SumVox

Official Homebrew tap for [SumVox](https://github.com/musingfox/SumVox) - Intelligent voice notifications for AI coding tools.

## Installation

```bash
brew tap musingfox/sumvox
brew install sumvox
```

## Upgrade

```bash
brew upgrade sumvox
```

## About SumVox

SumVox transforms AI coding sessions into voice notifications using:
- Multi-LLM support (Google Gemini, Anthropic Claude, OpenAI GPT, Ollama)
- Multi-TTS engines (Google TTS, macOS say)
- Automatic provider fallback
- 7ms startup time

For more information, visit the [main repository](https://github.com/musingfox/SumVox).
